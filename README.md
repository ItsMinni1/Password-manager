OVERVIEW

Password Manager is an application designed to securely store, manage, and retrieve user credentials such as usernames and passwords. The project focuses on demonstrating fundamental concepts of security, encryption, and data storage. It is intended primarily for learning and academic use.

The application helps users avoid insecure password practices and reduces the risks associated with storing sensitive information in plain text.

FEATURES

Secure storage of user credentials

Add new passwords

View saved credentials

Update existing entries

Delete stored passwords

Two-factor Authentication using a TOTP seed to be added in an authenticator application such as Google Authenticator

Encryption of passwords before storage

Interactive User Interface

INSTALLATION

Clone the repository:

git clone https://github.com/ItsMinni1/Password-manager.git

cd Password-manager

Install dependencies if required:

pip install -r requirements.txt

or

npm install

USAGE

Run the application using the appropriate command, for example:

python main.py

Once the program is running, users can:

Add new credentials

View saved passwords 

Modify existing entries

Remove stored credentials

SECURITY NOTES

Passwords are not stored in plain text.

Encryption is used to protect sensitive data.

This project is intended for educational purposes and is not recommended for production use without further security improvements.

LIMITATIONS

No cloud synchronization

No multi-user support

Limited authentication features

Not suitable for production environments
